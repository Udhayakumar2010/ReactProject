# React Counter App

A simple React application that demonstrates component-based architecture, state management using hooks, and reusable UI logic.

## 🚀 Features

* Functional components
* useState hook implementation
* Event handling in React
* Dynamic UI updates
* Clean folder structure

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite

## 📂 Project Structure

counter-app/
│
├── src/
│   ├── components/
│   ├── App.jsx
│   └── main.jsx
│
├── public/
└── package.json

## ▶️ How to Run the Project

Step 1: Clone the repository

git clone https://github.com/Udhayakumar2010/ReactProject.git

Step 2: Navigate into project folder

cd ReactProject

Step 3: Install dependencies

npm install

Step 4: Start development server

npm run dev
